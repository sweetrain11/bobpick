# 밥픽 — 오늘 뭐 먹지?

식당을 검색하고, 후보 메뉴를 직접 고른 뒤, 다섯 가지 랜덤 게임으로 오늘의 메뉴를 결정하는 반응형 웹서비스입니다.

## 주요 기능

- 키워드 기반 식당·메뉴 검색
- 식당별 메뉴 후보 추가·제외 및 전체 선택·해제
- 사다리타기, 제비뽑기, 복권 긁기, 룰렛, 슬롯머신
- `crypto.getRandomValues` 기반의 균등한 메뉴 선택
- 결과 확인, 같은 게임 재시도, 다른 게임 선택
- 모바일·데스크톱 반응형 UI와 키보드 접근성

## 로컬 실행

```bash
npm install
npm run dev
```

브라우저에서 `http://localhost:3000`을 엽니다.

## 배포

Next.js App Router 기반 프로젝트로 Vercel에서 바로 배포할 수 있습니다.

## 현재 데이터 범위

MVP 검증을 위해 4개 식당과 메뉴 샘플을 프런트엔드에 포함했습니다. 실제 서비스 전환 시 식당 검색 API 또는 자체 데이터베이스 연동이 필요합니다.
