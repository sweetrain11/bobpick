# 메뉴 추천 검색 사이트 기획 초안

> 문서 상태: 초기 아이디어 정리 단계  
> 앞으로 추가되는 요구사항에 맞춰 계속 확장하고 수정한다.

## 1. 프로젝트 개요

키워드 기반 음식점 검색을 통해 사용자가 원하는 음식점을 탐색하고, 사다리타기, 제비뽑기, 복권 긁기, 룰렛, 슬롯머신 같은 랜덤 게임을 통해 메뉴를 선택하도록 도와주는 웹사이트를 만든다.

## 2. 프로젝트 목적

- 사용자가 입력한 키워드를 기반으로 원하는 음식점을 쉽고 편리하게 검색할 수 있도록 돕는다.
- 무엇을 먹을지 고민하는 사용자가 메뉴를 결정하는 데 드는 부담을 줄인다.
- 음식점 검색 기능에 여러 형태의 랜덤 게임을 결합해 메뉴 선택 과정을 즐길 수 있게 한다.
- 사용자가 게임을 시작하기 전에 전체 메뉴에서 원하는 메뉴를 후보에 넣거나 빼며 선택 범위를 직접 조정할 수 있게 한다.
- 단순히 메뉴를 추천하는 데서 끝나지 않고, 사용자가 게임에 직접 참여하며 메뉴를 결정하는 경험을 제공한다.

## 3. 핵심 방향

### 검색 중심

사용자가 키워드를 입력해 원하는 음식점을 찾고 탐색할 수 있는 검색 기능을 사이트의 기반으로 삼는다.

### 메뉴 후보 직접 구성

사용자가 게임을 시작하기 전에 음식점의 전체 메뉴를 확인하고, 게임에 사용할 후보 메뉴를 직접 넣거나 뺄 수 있게 한다.

### 재미있는 메뉴 선택

사다리타기, 제비뽑기, 복권 긁기, 룰렛, 슬롯머신 같은 참여형 랜덤 게임을 이용해 메뉴를 재미있게 선택할 수 있게 한다.

### 선택 과정의 흥미

메뉴 추천 결과뿐만 아니라, 결과가 정해지는 과정에서도 기대감과 재미를 느끼도록 구성한다.

## 4. 현재 구상한 주요 기능

1. **키워드 기반 음식점 검색**
   - 사용자가 키워드를 입력해 원하는 음식점을 검색하고 탐색할 수 있는 기능

2. **게임 전 후보 메뉴 구성**
   - 음식점의 전체 메뉴를 확인할 수 있는 기능
   - 게임을 시작하기 전에 특정 메뉴를 후보 목록에 넣거나 뺄 수 있는 기능

3. **랜덤 게임형 메뉴 선택**
   - 사용자가 구성한 후보 메뉴 중 하나를 랜덤 게임을 통해 선택하는 기능
   - 제공할 게임
     1. 사다리타기
     2. 제비뽑기
     3. 복권 긁기
     4. 룰렛
     5. 슬롯머신

## 5. 기본 이용 흐름

1. 사용자가 사이트에 접속한다.
2. 키워드를 입력해 원하는 음식점을 검색하고 탐색한다.
3. 음식점의 전체 메뉴를 확인한다.
4. 게임에 사용할 후보 메뉴를 넣거나 뺀다.
5. 원하는 랜덤 게임을 선택하고 참여한다.
6. 사이트가 선택된 메뉴를 보여주고 사용자는 결과를 확인한다.
